Introduction

Java is a cross-platform, object-oriented programming language. It is known for being robust, secure, and designed to have as few implementation dependencies as possible. Java applications are typically compiled to bytecode that can run on any Java Virtual Machine (JVM), making it a truly cross-platform language. Java contains a rich standard library that includes classes for data structures, networking, utilities, graphics, and much more. It has a strong foundation with core language elements such as operators, control structures, and statements. Java can be extended for various use cases by including additional libraries and frameworks; for example:

  • Client-side Java applications can be developed using frameworks like Swing or JavaFX, allowing developers to create graphical user interfaces (GUIs) that run on desktop environments.
  • Server-side Java is widely used in enterprise environments. With frameworks like Spring and Java Enterprise Edition (Java EE), you can build web applications, services, and handle backend logic with ease.

This guide assumes you have the following basic background:

  • A general understanding of programming principles.

Java and JavaScript

Java and JavaScript are similar in name but fundamentally different. Java is a statically-typed, compiled language, whereas JavaScript is dynamically typed and interpreted. While Java follows a class-based object model, JavaScript supports prototype-based inheritance. Despite these differences, both languages are versatile in their respective domains.
Java is known for its:

  • Static typing: Variables must be declared with a data type, and their types are checked at compile-time.
  • Class-based object model: Java relies on class declarations for creating objects and enabling inheritance.
  • Strong typing: Data types must be explicitly declared, and improper usage of types will cause compile-time errors

JavaScript, on the other hand, is loosely typed, dynamic, and does not require explicit declarations for variables, methods, or classes.

Hello World Example

To get started with writing Java, create a simple class and a main method in a text editor or IDE. This is how you would write a "Hello World" program in Java: 

public class HelloWorld { public static void main(String[] args) {
   System.out.println("Hello, World!"); } }

Compile the program using the command javac HelloWorld.java, and then run it with the command java HelloWorld. You should see "Hello, World!" printed in the console.

Variables

You use variables to store data in Java programs. Java is a statically-typed language, meaning that variables must be declared with a specific data type.

Declaring Variables

In Java, you declare a variable by specifying the data type followed by the variable name. For example: 

int x = 42; // Declares an integer variable
String name = "John"; // Declares a String variable

Java supports different data types like int, double, char, boolean, and String. The variables must be initialized before use, and Java enforces strong typing.

Variable Scope

In Java, variables can be classified based on their scope:

  • Local variables: Declared inside methods or constructors, these are accessible only within that block of code
  • Instance variables: Declared inside a class but outside any method, these variables are accessible by all methods of the class.
  • Static variables: Declared with the static keyword, these belong to the class rather than any specific instance, meaning all instances share this variable. 
public class ScopeExample {
  static int globalVariable = 100; // Static variable

  public void method() {
    int localVariable = 10; // Local variable
    System.out.println(localVariable);
  }
}
Constants

In Java, constants are declared using the final keyword. Once a variable is declared final, its value cannot be changed during the program's execution. Example: 

double PI = 3.14159;

The PI constant cannot be reassigned a new value once it's declared.

Data Types

Java supports both primitive data types and reference types (like objects).

  • Primitive types: These are the basic types built into the language, such as:
    • int (integer numbers),
    • double (floating-point numbers),
    • char (single character),
    • boolean (true/false).
  • Reference types: These refer to objects and include arrays, classes, interfaces, and more.
    For instance: 
    String name = "Java";
if...else Statement

Java provides the traditional if and else control flow for conditional execution:

if (condition) { // code to execute if the condition is true } else { // code to execute if the condition is false }

For example:

int age = 18;
if (age >= 18) {
  System.out.println("You are an adult.");
} else {
  System.out.println("You are a minor.");
}
while Statement

The while loop in Java executes a block of code as long as the specified condition remains true. 

int count = 0;
while (count < 5){
 System.out.println("Count is: " + count);
 count++;
}

This will print the value of count from 0 to 4.

Function Declarations

In Java, functions are typically called methods and are defined inside classes. A method consists of:

  • Return type: The type of value the method returns.
  • Method name: A unique name to identify the method.
  • Parameter list: Input values passed to the method.

For example, a method that squares a number looks like this:

public int square(int number) {
  return number * number;
}

This method returns the square of the input number.
